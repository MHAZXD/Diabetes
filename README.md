# Diabetes Prediction using Machine Learning (SVM)

## Project Overview
This project builds a Machine Learning model to predict whether a patient is diabetic or non-diabetic using the PIMA Indians Diabetes Dataset.

The project demonstrates end-to-end implementation of:
- data loading
- preprocessing
- model training
- evaluation using SVM classifier

---

## Dataset

The dataset used in this project is the PIMA Indians Diabetes Dataset.

[Download Dataset (Included in this Repository)](https://raw.githubusercontent.com/MHAZXD/Diabetes/main/diabetes.csv)

[Original Source on Kaggle](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database)

The dataset contains 8 medical features and a target `Outcome` for diabetes diagnosis.

---

##  Objective
To develop a binary classification model that predicts:
- 0 → Non-Diabetic
- 1 → Diabetic

---

##  Technologies Used
- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Google Colab  

---

##  Project Workflow
1. Import dependencies  
2. Load and analyze dataset  
3. Separate features and target  
4. Apply feature scaling (`StandardScaler`)  
5. Train-test split  
6. Train Support Vector Machine (SVM) model  
7. Evaluate performance using accuracy score  

---

##  Model
**Support Vector Machine (SVM)** is used for binary classification.

---

##  How to Run
1. Clone this repository --> https://github.com/MHAZXD/Diabetes.git
2. Install required libraries: pip install numpy pandas scikit-learn
3. Open and run the notebook in Google Colab or Jupyter.
